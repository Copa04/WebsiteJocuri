Limbajul HTML

HTML (HyperText Markup Language) este un limbaj de marcare utilizat pentru a crea pagini web.

Un exemplu de cod html îl găsiți în imaginea următoare care se află la adresa https://upload.wikimedia.org/wikipedia/commons/e/e4/HTML_logo.png.
Imaginea are atributul height de valoare 147.

Tag-urile html cele mai utilizate în realizarea tabelelor sunt:

table definește un tabel
tr un rând într-un tabel
td o celulă într-un tabel

Observație: Pentru a pune un contur unui tabel se poate (dar nu se recomanda) să se puna atributul border="1".
La validare va da eroarea: The border attribute on the table element is obsolete.

Formularele se pot crea cu ajutorul tag-ului form care poate contine tag-urile: input, textarea, label, select, option, button.
Pentru mai multe detalii consultati cursul sau tutorialul HTML Forms and Inputs de la adresa http://www.w3schools.com/html/html_forms.asp.

Pentru a crea liste se pot folosi tag-urile:

  1. ul - lista neordonata
  2. ol - lista ordonata
  3. li - item din lista
  4. dl - lista cu descrieri
  5. dt - termen dintr-o lista cu descrieri
  6. dd - descrierea dintr-o lista cu descrieri

Pentru a insera un document HTML in altul se poate folosi tag-ul <iframe>.
Astfel, exemplul de mai jos insereaza continutul de la adresa https://html.spec.whatwg.org/multipage/.
